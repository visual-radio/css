클래스 선택자 작성간 주의 사항

1. 클래스 명의 첫글자는 숫자 불가(단, 영문 이후 연결된 수자는 관계 없음, 특수문자 대부분 사용 불가, 언더바 제외하고 되도록 쓰지 말것)

2. 클래스 명은 복수의 클래스명으로 작성이 가능(class="클래스명1 클래스명2")

3. 클래스 명 작성시 의미있는 이름을 부여하는 것이 협업에 유리함

4. 클래스 명으로 단어와 단어를 연결시, '_' (언더바) 사용 (스네이크 기법 class="word_word") (파스칼 기법 class="WordWord") (카멜 기법 class="wordWord")

5. 클래스 명은 재활용이 가능

아이디 선택자 작성간 주의사항

1. 아이디 선언시, 첫글자는 숫자 불가(단, 영문 이후 연결된 수자는 관계 없음, 특수문자 대부분 사용 불가, 언더바 제외하고 되도록 쓰지 말것)

2. 복수의 아이디명 불가

3. 의미있는 이름을 부여할 것

4. 단어와 단어의 연결은 클래스명과 동일

5. 아이디 명은 재활용 불가능(css에서는 허영되나 js에서는 불허)

6. 아이디 명은 구조상 규모가 큰곳에 적절히 활용하면 문서상에서 찾기가 쉬움
